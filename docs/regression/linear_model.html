最佳線性不偏預測（Best Linear Unbiased Estimator, BLUE）​

Q: 最小二乘法總是能得到一個好的 β^~\utilde{\hat{\beta}}β^​​ 嗎？或者說，在什麼情況下最小二乘法難以得到一個好的 β~\utilde{\beta}β​？

  1. 在下圖情況中，如果我們計算殘差 eie_iei​，雖然可能會得到和為 0 的結果，但殘差的分佈有很明顯的正負聚集現象。而我們希望殘差的正負隨機分佈的。

    為了避免這種情況，我們需要誤差的期望為 0，即 E[ε~]=0E[\utilde{\varepsilon}]=0E[ε​]=0。

  2. 下圖中的數據，隨著 XXX 的增加，數據也越分散。

    為了避免這種情況，我們需要所有誤差的方差是一樣的，即 Var[e~]=σ2Var[\utilde{e}]=\sigma^2Var[e​]=σ2，其中 σ2\sigma^2σ2 是一個常數。

  3. 當沒有按照正規的程序獲取數據時，可能會出現下圖的情況。各個數據之間可能存在著相互關聯的情況。那麼擬合出來的回歸線會存在誤導性。

    為了避免這種情況，我們需要所有誤差之間是沒有線性關係的，即 Cov[ei,ej]=0,∀i≠jCov[e_i, e_j]=0, \forall i\neq jCov[ei​,ej​]=0,∀i=j。
