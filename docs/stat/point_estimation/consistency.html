一致性（Consistency）

當我們獲得的數據量越來越多時，一個好的估計方法應該要能更準確的估計出真實的數據分布。這種性質稱為一致性。

Definition

∀ε>0\forall \varepsilon >0∀ε>0 ，we say δn→n→∞Pη(θ)\delta_n \xrightarrow[n\to\infty]{P} \eta(\theta)δn​Pn→∞​η(θ) if

Pθ(∣δn−η(θ)∣≤ε)→n→∞1P_\theta(|\delta_n-\eta(\theta)|\le \varepsilon) \xrightarrow[n\to\infty]{} 1Pθ​(∣δn​−η(θ)∣≤ε)n→∞​1

也就是 δn\delta_nδn​ 以幾率收斂到 η(θ)\eta(\theta)η(θ) 。

Theorem

讓 δn\delta_nδn​ 是 η(θ)\eta(\theta)η(θ) 的一致估計，則對於任何符合以下要求的數列 an,bna_n, b_nan​,bn​：

lim⁡n→∞an=1lim⁡n→∞bn=0\lim_{n\to\infty} a_n = 1 \quad\lim_{n\to\infty} b_n = 0n→∞lim​an​=1n→∞lim​bn​=0
