EMS rule

EMS rule for balanced designs​

for model

Yijk=μ+Ai+Bj+ABij+ϵijkY_{ijk} = \mu + A_i + B_j + AB_{ij} + \epsilon_{{ij}k}Yijk​=μ+Ai​+Bj​+ABij​+ϵijk​

i=1,⋯ ,a,j=1,⋯ ,b,k=1,⋯ ,n,ε(ij)k∼iidN(0,σ2)i=1,\cdots,a,\quad j=1,\cdots,b,\quad k=1,\cdots,n,\quad\varepsilon_{(ij)k}\overset{\text{iid}}{\sim}N(0,\sigma^2)i=1,⋯,a,j=1,⋯,b,k=1,⋯,n,ε(ij)k​∼iidN(0,σ2)

  1. 準備一張表格，row 為所有 factor 、交互作用和隨機項目，column 為所有下標及其對應的 factor 是隨機或固定。

接下來根據以下規則計算 EMS：

  1. 忽略該 trt 下標中，括號之外的所有 column （e.g. AiA_iAi​ 忽略 iii col，ε(ij)k\varepsilon_{(ij)k}ε(ij)k​ 忽略 kkk col）。
  2. 找到所有包含該 trt 下標的所有 row，將其對應的 column 值相乘。
  3. 篩選出的 row 中，如果包含隨機效應，則其對應的方差為 στ\sigma_\tauστ​，如果都是固定效應，則其對應的方差為 ϕσ\phi_\sigmaϕσ​。
  4. 將 2. 和 3. 的結果相乘，並將每個 row 的結果相加，即為 EMS。

Remark: 如果 EMS rule 無法給出 effect 的檢定方法。可以用以下兩種方式：

  1. 假設一些 effect/intraction 為 0 。
  2. 用漸進方法得到 F-test。

Remakr: ε(ij)k\varepsilon_{(ij)k}ε(ij)k​ 與 εijk\varepsilon_{ijk}εijk​ 兩種符號的意義是不同的。前者表示 kkk 在 ijijij 效應下得到，意味著獲得數據的 trt 是隨機出現的，後者則代表數據是輪流獲得的。
